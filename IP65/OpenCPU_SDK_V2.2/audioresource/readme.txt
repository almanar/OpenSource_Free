Build audio stream data

1. edit audio.txt, add all audio files to audio.txt.
	example: add "....audioresource\audio\wav11k.wav" to audio.txt.

2. Run "QlAudioResGen.exe", which will build all audio files to the audio stream data in audio.txt.

3. QlAudioResGen.exe will create resource_audio.c and resource_audio.h, then copy these files to project.
